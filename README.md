# AmandaTheAdventurer3-ru

Русский перевод для игры Amanda The Adventurer 3, предназначенный для использования с XUnity.AutoTranslator.

## Установка

1. **Скачайте и установите** XUnity.AutoTranslator:
   - Последняя версия: [v5.5.0](https://github.com/bbepis/XUnity.AutoTranslator/releases/tag/v5.5.0)

2. **Установите файлы перевода**:
   - Скопируйте все файлы из этого репозитория в папку:
   ```
   BepInEx/Translations/ru/
   ```

## Настройка конфигурации

Создайте/отредактируйте файл `BepInEx/config/AutoTranslatorConfig.ini` со следующим содержимым:

```ini
[Service]
Endpoint=GoogleTranslate
FallbackEndpoint=GoogleTranslate

[General]
Language=ru
FromLanguage=en
OnlineMappingHint=1
EnableSSL=1

[Files]
Directory=Translation\{Lang}\Text
OutputFile=Translation\{Lang}\Text\_AutoGeneratedTranslations.txt

[TextFrameworks]
EnableIMGUI=True
EnableUGUI=True
EnableNGUI=True
EnableTextMeshPro=True
EnableTextMesh=True

[Behaviour]
MaxCharactersPerTranslation=1000
EnableSilentMode=False
SkipAlreadyTranslatedText=0
AutoTranslateUI=1
UseStaticTranslations=False
GeneratePartialTranslations=True
EnableBatching=False
MinDialogueChars=1
TranslationDelay=0.02
MaxTranslationWaitTime=2
EnableUIResizing=True
CacheParsedTranslations=True

[Texture]
EnableTextureTranslation=False

[ResourceRedirector]
EnableTextAssetRedirector=True

[Http]
DisableCertificateValidation=False
Timeout=3
MaxRetries=1
```

## Особенности

- Полный перевод текстовых элементов игры на русский язык
- Оптимизированные настройки для стабильной работы
- Поддержка всех текстовых систем игры (IMGUI, UGUI, TextMeshPro и др.)
- Автоматическое применение переводов к интерфейсу

Я не являюсь автором этого перевода и он машинный!

## Примечания

- Убедитесь, что у вас установлен .NET Framework 4.7.2 или выше
- Для работы онлайн-перевода требуется стабильное интернет-соединение
- Первый запуск может занять больше времени из-за генерации переводов
